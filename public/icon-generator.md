# PWA Icon Generator

## Required Icons

You need to create two icon files:
- `icon-192x192.png` (192x192 pixels)
- `icon-512x512.png` (512x512 pixels)

## Quick Options

### Option 1: Use an Online Generator
1. Go to https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator
2. Upload your logo or create an icon
3. Download the generated icons
4. Place them in the `/public` folder

### Option 2: Create Simple Icons
You can create simple icons with:
- Background: Gradient from indigo-600 (#4f46e5) to purple-600 (#9333ea)
- Text: "K" in white, bold, centered
- Rounded corners (optional)

### Option 3: Use Placeholder (For Testing)
For now, you can use any 192x192 and 512x512 PNG images as placeholders.

## Icon Requirements
- Format: PNG
- Sizes: 192x192 and 512x512 pixels
- Background: Should work on both light and dark backgrounds
- Purpose: Should be recognizable at small sizes
